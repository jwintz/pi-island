Pi Island is a native macOS application built with Swift and SwiftUI.

## High-Level Architecture

```
Pi Island (macOS app)
    |
    |--- stdin/stdout JSON protocol
    |
    v
pi --mode rpc (child process)
```

## Core Components

- **PiRPCClient**: Actor-based RPC client
- **SessionManager**: Multi-session management
- **NotchViewModel**: State management for UI
- **NotchWindowController**: NSWindowController for panel

## File Structure

```
Sources/PiIsland/
├── Core/           # Geometry, events, view models
├── UI/             # SwiftUI views
└── RPC/            # Protocol types, client, sessions
```
