## Requirements

- macOS 14.0 or later
- Xcode 15.0 or later

## Development Build

```bash
swift build && .build/debug/PiIsland
```

## App Bundle

```bash
./scripts/bundle.sh
```

Creates `Pi Island.app` with proper bundle structure.

## Packaging Options

```bash
./scripts/bundle.sh --sign --dmg
```
