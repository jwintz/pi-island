## Prerequisites

Before installing Pi Island, ensure you have:

1. **macOS 14.0 or later**
2. **Pi coding agent** installed and accessible via the `pi` command
3. A MacBook with notch (recommended, but any Mac works)

## Download

Download the latest release from the [GitHub Releases](https://github.com/jwintz/pi-island/releases) page.

## Install from DMG

1. Open `Pi-Island-x.x.x.dmg`
2. Drag `Pi Island.app` to the Applications folder
3. Eject the disk image

## First Launch

1. Open `Pi Island.app` from Applications
2. If you see a security warning, go to System Settings > Privacy & Security
3. The app will appear in your notch area

## Auto-Launch at Login

1. Open Pi Island settings (gear icon)
2. Enable "Launch at Login"

## Troubleshooting

### Pi Island doesn't appear

- Ensure the app is running (check Activity Monitor)
- Try moving your mouse to the top center of the screen

### No sessions visible

- Verify Pi is installed: `which pi`
- Check that Pi has sessions: `ls ~/.pi/agent/sessions/`
