## In-App Settings

Access settings by clicking the gear icon:

- **Launch at Login**: Start automatically when you log in
- **Show in Dock**: Display dock icon (not recommended)

## File Locations

| Path | Purpose |
|------|---------|
| `~/.pi/agent/sessions/` | Pi session files (JSONL) |
| `~/.pi/agent/auth.json` | Pi authentication |
| `~/.pi-island-debug.log` | Debug log (when enabled) |

## Debug Logging

View system logs:

```bash
log show --predicate 'subsystem == "com.pi-island"' --last 60s --info
```
